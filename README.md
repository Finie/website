# Grace Kamau Portfolio

A polished static portfolio website for **Grace Kamau**, built with **React + Vite + TypeScript**.

## Run locally

```bash
npm install
npm run dev
```

The local dev server will print the URL in the terminal, typically `http://localhost:5173`.

## Build for production

```bash
npm run build
npm run preview
```

## Highlights

- Single-page recruiter-friendly portfolio
- Responsive layout for desktop, tablet, and mobile
- Sections covering frontend, backend, database, and project work
- Static deployment friendly with no backend or CMS required
